#Simple Ripple Animation

Simple and customizable ripple animation for flutter app!

## Installation

Use the package manager [pub](https://pub.dev/) to install rippleanimation.


## Usage


```dart
RippleAnimation(
  repeat: false,
  color: Colors.blue,
  minRadius: 100,
  ripplesCount: 6,
  child: Container()
),
```
